
En requeriment.txt se puede encontrar lo necesario de instalar

Para correr la app:
1 - Se debe realizar la instalacion de waitress con el comando :

                    pip install waitress

2 - Se escribe en la terminal el comando (ubicandose en la carpeta tp):

                    waitress-serve --port=5000 app:app

Para crear la base de datos:
Una vez corriendo el programa se debe ejecutar la ruta :

                    localhost:5000/init

En la carpeta del proyecto se encuentran :

1- Un documento word con los EndPoints.
2- Archivo de postman exportado.
3- Base de datos exportada.
